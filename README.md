## Como utilizar o projeto

Siga estes passos para instalar o projeto na sua maquina.

- `git clone https://github.com/jonatan-abade/desafio_3lm`
- `cd desafio_3lm`
- `cp .env.example .env`
- `composer install`
- `npm install`
- `npm run dev`
- `npm run dev`
- `php artisan key:generate`
- `php artisan migrate`
- `php artisan serve`
